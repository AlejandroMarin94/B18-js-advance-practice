# Practica del bootcamp Code Space basada en recrear una pagina de consultas sobre series y peliculas #
## Tecnologias usadas ##
## Tecnologías utilizadas

- **JavaScript (ES6+)**: para la lógica principal y el manejo de promesas.
- **Fetch API**: para realizar peticiones HTTP y consumir datos externos.
- **JSON**: como formato de intercambio de datos entre cliente y servidor.
- **Vite**: como entorno de desarrollo rápido y moderno.
- **SCSS**: para la maquetación y estilos de la interfaz.
